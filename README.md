# 🚀 DepoFly: Akıllı Stok ve Envanter Yönetim Sistemi

DepoFly, modern bir depo yönetiminden beklenen tüm ihtiyaçları karşılayan, hız, güvenlik ve kullanıcı deneyimi odaklı geliştirilmiş bir ASP.NET Core MVC projesidir.

## ✨ Öne Çıkan Özellikler
* **Akıllı Barkod Tarama:** Donanım gerektirmeden mobil kamera üzerinden hızlı ürün kontrolü.
* **Haptic Feedback:** Başarılı işlemlerde mobil cihazlarda titreşimli geri bildirim (Vibration API).
* **Gelişmiş Yetkilendirme:** RBAC mimarisi ve OAuth 2.0 (Google/Facebook Login) desteği.
* **Modern UI:** Glassmorphism tasarımı ile şık ve minimalist arayüz.
* **Güvenlik Odaklı:** Sunucu seviyesinde Fail2Ban koruması ve uçtan uca HTTPS.

## 🛠️ Teknik Stack
* **Backend:** ASP.NET Core MVC, Entity Framework Core, Identity
* **Database:** MS SQL Server
* **Frontend:** Bootstrap 5, jQuery, SweetAlert2, DataTables, Cropper.js
* **Performans:** In-Memory Caching

## 🚀 Kurulum
1. Projeyi klonlayın: `git clone https://github.com/mustafaburaksahin/DepoFly.git`
2. `appsettings.json` dosyasındaki ConnectionString ve API anahtarlarınızı düzenleyin.
3. Package Manager Console üzerinden `Update-Database` komutunu çalıştırın.
4. Projeyi ayağa kaldırın!
